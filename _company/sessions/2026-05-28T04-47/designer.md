# 🎨 Designer — 새로운 비전과 정체성을 시각적으로 담아낼 브랜드 컬러 팔레트, 로고/워터마크 업데이트 방향, 그리고 모든 플랫폼(웹, 유튜브, 인스타)에 통일적으로 적용될 타이포그래피 가이드라인을 설계하시오.

🎨✨ **새로운 브랜드 아이덴티티(BI) 구축 계획서** ✨📊

**목표:** '감성적 접근'에서 벗어나 '신뢰 기반의 데이터 중심'의 전문성을 갖춘, 시장을 선도하는 부동산 자문사 이미지 확립.

**핵심 키워드:** **신뢰(Trust) > 전문성(Expertise) > 데이터 기반(Data-Driven)**

---

### 1. 컬러 팔레트 (Color Palette) 재정의

기존의 따뜻하고 감성적인 색상(골드, 베이지)을 지양하고, 신뢰감과 지성을 상징하는 차분하고 구조적인 색상으로 전환합니다.

| 색상명 | HEX Code | 역할 및 의미 | 적용 예시 |
| :--- | :--- | :--- | :--- |
| **메인 컬러 (신뢰)** | `#003366` (딥 네이비) | 가장 핵심적인 컬러. 신뢰, 전문성, 안정감, 지성을 상징. (금융/컨설팅 업계 선호) | 로고 타입, 메인 헤드라인, 배경 강조색. |
| **서브 컬러 (명료성)** | `#4D90FE` (스카이 블루) | 데이터 시각화, 강조 포인트, 통찰력을 상징. (기술력, 명확성) | 그래프 라인, 핵심 데이터 강조, CTA 버튼. |
| **보조 컬러 (균형)** | `#CCCCCC` (라이트 그레이) | 배경, 텍스트의 가독성 확보. (모던함, 깔끔함) | 배경색, 구분선, 보조 텍스트. |
| **강조 컬러 (액션)** | `#D9534F` (버건디/딥 레드) | 경고, 중요 변화, 혹은 강력한 액션 유도. (신중함, 주목도) | 리스크 경고, 핵심 발견점 강조. |

---

### 2. 타이포그래피 (Typography) 표준화

가독성과 구조적 명확성을 최우선으로 고려합니다. 복잡한 세리프체 지양.

*   **메인 폰트 (제목/헤드라인):** **Noto Sans KR** 또는 **Pretendard**
    *   *선택 이유:* 모던하고 깔끔한 산세리프체로, 디지털 환경과 인쇄물 모두에서 명확한 가독성을 제공합니다.
*   **서브 폰트 (본문):** **Noto Sans KR** (기본 크기 14pt 기준)
    *   *적용 원칙:* 폰트 크기 변화에 따른 위계질서(H1 > H2 > Body)를 명확히 하여, 정보의 계층 구조를 시각적으로 인지하게 합니다.

---

### 3. 로고 및 심볼 전략 (Logo & Symbol)

*   **전략:** 추상적이거나 장식적인 요소 배제. '데이터의 연결', '구조화', '전망(View)'의 개념을 기하학적 형태로 표현.
*   **심볼 아이디어:** 네 개의 점이 연결되어 하나의 안정적인 구조(사각형 또는 육각형)를 이루는 형태. (연결성 + 안정성)
*   **적용 예시:** 심볼과 텍스트(회사명)를 나란히 배치하며, 심볼을 메인 컬러(딥 네이비)로 통일.

---

### 4. 채널별 적용 가이드라인 (Application Guidelines)

| 채널 | 디자인 목표 | 적용 요소 | 예시 |
| :--- | :--- | :--- | :--- |
| **웹사이트/보고서** | 정보의 위계화 및 신뢰감 구축 | 딥 네이비 배경 섹션 사용, 데이터 시각화에 스카이 블루 활용. | 섹션 구분 시 얇은 그레이 라인 사용. |
| **프레젠테이션** | 논리적 흐름과 설득력 강조 | 슬라이드 마스터에 네이비/블루 계열의 그라데이션을 은은하게 적용. | 발표자 노트(Speaker Note)는 깔끔한 그레이 배경 사용. |
| **소셜 미디어 (SNS)** | 간결함과 임팩트 | 텍스트 위주 콘텐츠에 컬러 블록(딥 네이비 배경 + 흰색 텍스트)을 활용하여 가독성을 극대화. | 인포그래픽 제작 시, 데이터 라인에 스카이 블루를 일관되게 사용. |
